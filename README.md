PHP Code Quality Analyzer
=========
Check out and fix your PHP code ( PSR-2, PSR-1, Zend, PEAR, Squiz and PHPCS Standards )

Installation
======================
Clone PHP-Code-Quality-Analyzer in your project root directory and run : 

composer install 


Requirements
======================

* [Composer] [1]
* [PHP-CS-Fixer] [2]
* [PHP-Codesniffer] [3]
* [PHPMD] [4]
* [PHPMetrics] [5]
