PHP Code Quality Analyzer
=========
check out and fix your PHP code ( PSR-2 coding standar )

Installation
======================
clone the project in your project root directory and run : composer install 

Requirements
======================

* [Composer] [1]
* [PHP-CS-Fixer] [2]
* [PHP-Codesniffer] [3]
* [PHPMD] [4]
* [PHPMetrics] [5]
